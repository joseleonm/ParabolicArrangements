# Parabolic Arrangements - Cohomology Ring Computation

[![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/joseleonm/ParabolicArrangements/HEAD?labpath=notebooks%2Fdemo_parabolic_dga.ipynb)

[![SageMath](https://img.shields.io/badge/SageMath-10.7+-blue.svg)](https://www.sagemath.org/)

SageMath implementation of differential graded algebras for computing the cohomology ring of real parabolic arrangements.

## Features

- Compute Betti numbers and cohomology bases for parabolic arrangements
- Fast cup product computation via precomputed geometric structure (~720x speedup)
- Automated verification that H² is generated by products of H¹
- Comprehensive DGA structure validation (Leibniz rule)

### Basic Usage

```python
sage: load('parabolic_dga.sage')
sage:
sage: # Build A₅ (3-equal) arrangement
sage: W, Plist, _ = build_W_P('A', 5)
sage: Delta = ideal_k_parabolic(W, Plist, k=3)
sage: dga = ParabolicZonotopalDGA(W, Plist, Delta)
sage:
sage: # Verify H² is generated by H¹ products
sage: result = dga.cup_product_span_analysis(ring=GF(2))

result = dga.cup_product_span_analysis(ring=GF(2))

```

## Mathematical Background

This implementation provides a cellular DGA model for computing cohomology rings of complements of real parabolic arrangements. The key innovation is using the **zonotopal cup product** with **precomputed geometric structure** to avoid redundant Coxeter group computations.

For theoretical background, see:
- J. Cantarero and J. L. León-Medina, "The Cohomology Ring of Real Parabolic Arrangements", manuscript in preparation
- A. Björner and G. Ziegler, "Combinatorial stratification of complex arrangements", J. Amer. Math. Soc. 5 (1992), 105–149
